Story Crafter Game

Project Addendum:

In this project, I created a web server using Express.js to generate a simple Mad Libs story. Here's a breakdown of my process:

  • Form Structure: The user is now asked to provide a singular noun, adjective, verb (past tense), adverb, plural noun, and verb for the story generation.
  • Routing: The server uses two routes: one to show the form and another to handle form submissions and generate the Mad Lib story.
  • Story Generation: The server dynamically inserts the user input into a predefined story template, providing a unique, fun story for each submission.
